🛠️ Engineering Tools

Direct access to hidden factory menus, picture-quality (PQ) settings and system activities on the XBJ A5 Pro. For advanced users only — some menus can change low-level display calibration.
Native app required
These shortcuts only work inside the Projector Guy app installed on your projector — they won't do anything in a regular web browser.
✅ Confirmed working shortcuts
These were tested over ADB and are known to work on this firmware.
Browse any package
Enter a package name to list its activities, then tap one to launch it directly. Useful for exploring unfamiliar factory/service apps.
